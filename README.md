# Green Trees Initiative Website

## Overview

This project is a multi-page website for the Green Trees Initiative, a climate-focused organization supporting sustainable work for women in developing nations and addressing climate change.

The website consists of two main pages:
- Home Page (`index.html`): Introduces the cause, features initiatives, shares stories and news, and encourages users to act and donate.
- Donation Page (`donate.html`): Provides a donation form and information about the impact and security of donations.

## Features

- Mobile-first responsive design using flexbox and CSS grid.
- Accessibility enhancements including ARIA landmarks and roles.
- Common CSS stylesheet for styling consistency with structural separation.
- Clear navigation between pages.
- Focus on semantic HTML and clean, maintainable CSS.

## Usage

To view the website, open `index.html` or `donate.html` in a modern web browser.

## Development

- All styles are located in `css/styles.css`.
- Assets such as images are located in the `assets/` directory.
- Responsive design adapts layout for mobile, tablet, and desktops using media queries.
- Semantic HTML with accessibility attributes added.
